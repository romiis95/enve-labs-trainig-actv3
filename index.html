Pasos para ejecutar el código:

Abre el archivo en un navegador web compatible. Verás un cuadro de diálogo emergente que te solicitará ingresar un número o la letra 'q' para salir. Ingresa los números deseados uno por uno. Si deseas finalizar la entrada, ingresa la letra 'q'. Después de ingresar la letra 'q' o finalizar la entrada, se mostrará en la consola del navegador un array con todos los números ingresados.
